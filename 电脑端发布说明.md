# 电脑端发布说明（GitHub Pages + 自定义域名）

项目仓库：https://github.com/xynong/travel-share.git
自定义域名：xynong.ccwu.cc

## 1. 先把网站文件提交到 GitHub
在电脑终端中执行：

```powershell
cd C:\Users\xieyong\GitHub\travel-share
git add .
git commit -m "更新旅游官网"
git branch -M main
git push -u origin main
```

## 2. 在 GitHub 打开 Pages
1. 打开 GitHub 仓库页面
2. 点击 Settings
3. 左侧找到 Pages
4. Source 选择 Deploy from a branch
5. Branch 选择 main，Folder 选择 /root
6. 点击 Save

## 3. 绑定自定义域名
1. 在 Pages 页面中找到 Custom domain
2. 填写：
   xynong.ccwu.cc
3. 点击 Save

## 4. 配置域名解析
在你的域名服务商后台添加解析：
- 类型：CNAME 或 A 记录
- 主机名：xynong.ccwu.cc
- 目标：GitHub Pages 地址

## 5. 等待生效后访问
访问地址：
https://xynong.ccwu.cc
