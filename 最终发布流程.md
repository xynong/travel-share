# 最终发布流程（Windows 电脑版）

仓库地址：https://github.com/xynong/travel-share.git
自定义域名：xynong.ccwu.cc

## 1. 安装 Git
如果电脑没有 Git，请先安装：
https://git-scm.com/download/win

安装后打开 PowerShell，验证：
```powershell
git --version
```

## 2. 进入项目目录
```powershell
cd C:\Users\xieyong\GitHub\travel-share
```

## 3. 配置 Git 身份
```powershell
git config --global user.name "xieyong"
git config --global user.email "your@email.com"
```

## 4. 提交并推送到 GitHub
```powershell
git status
git add .
git commit -m "更新旅游官网"
git branch -M main
git push -u origin main
```

## 5. 在 GitHub 开启 Pages
1. 打开 GitHub 仓库页面
2. 点击 Settings
3. 左侧进入 Pages
4. Source 选择 Deploy from a branch
5. Branch 选择 main，Folder 选择 /root
6. 点击 Save

## 6. 绑定自定义域名
1. 在 Pages 页面中找到 Custom domain
2. 填写：
   xynong.ccwu.cc
3. 点击 Save

## 7. 配置域名解析
在你的域名服务商后台添加解析：
- 类型：CNAME 或 A 记录
- 名称/主机名：xynong.ccwu.cc
- 指向：GitHub Pages 地址

## 8. 访问网站
等待几分钟到几个小时后访问：
https://xynong.ccwu.cc

## 9. 最短版命令
```powershell
cd C:\Users\xieyong\GitHub\travel-share
git add .
git commit -m "更新旅游官网"
git push origin main
```
