# 手机端发布说明（GitHub Pages + 自定义域名）

域名：xynong.ccwu.cc
仓库：https://github.com/xynong/travel-share.git

## 1）开启 GitHub Pages
1. 打开 GitHub 仓库页面。
2. 点 Settings。
3. 找到 Pages。
4. Source 选 Deploy from a branch。
5. Branch 选 main，Folder 选 /root。
6. 点 Save。

## 2）绑定自定义域名
1. 在 Pages 页面里，Custom domain 填写：
   xynong.ccwu.cc
2. 点 Save。
3. 等待部署完成。

## 3）域名解析
在你的域名服务商里添加解析：
- 方式：CNAME 或 A 记录
- 目标：GitHub Pages 地址
- 域名：xynong.ccwu.cc

## 4）等待生效
等几分钟到几小时后访问：
- https://xynong.ccwu.cc

## 5）最短版
- 先把网站推到 GitHub
- 然后开启 Pages
- 填写域名 xynong.ccwu.cc
- 配置解析
- 等待生效
