# 最终最短操作清单

## 1. 在 GitHub 里做这几步
1. 打开仓库：https://github.com/xynong/travel-share
2. 点击 Settings
3. 左侧点 Pages
4. Source 选 GitHub Actions
5. 点击 Save
6. 在 Custom domain 里填：xynong.ccwu.cc
7. 点击 Save

## 2. 在域名服务商后台做这几步
1. 打开你的域名后台
2. 找到 DNS / 解析管理
3. 新增 4 条 A 记录：
   - 185.199.108.153
   - 185.199.109.153
   - 185.199.110.153
   - 185.199.111.153
4. 主机名/名称填 @
5. 保存

## 3. 等待生效后访问
- 默认地址：https://xynong.github.io/travel-share/
- 自定义地址：https://xynong.ccwu.cc
