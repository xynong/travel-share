# 网站上线最终操作

## 1. GitHub 仓库设置
打开：
https://github.com/xynong/travel-share/settings/pages

## 2. Pages 配置
- Source：选择 GitHub Actions
- 保存

## 3. 自定义域名
- Custom domain：填 xynong.ccwu.cc
- 保存

## 4. 域名解析
在你的域名服务商添加 4 条 A 记录：
- 185.199.108.153
- 185.199.109.153
- 185.199.110.153
- 185.199.111.153

如果你的域名托管在 Cloudflare，请把这 4 条记录设置为“DNS only（灰云）”，不要开启代理。
## 5. 等待生效
等待 5～15 分钟后访问：
- https://xynong.github.io/travel-share/
- https://xynong.ccwu.cc/
