# GitHub Pages 启用说明

仓库地址：https://github.com/xynong/travel-share

## 1. 进入设置
1. 打开仓库页面
2. 点击 Settings
3. 左侧选择 Pages

## 2. 启用 Pages
- Source：GitHub Actions
- 点击 Save

## 3. 绑定自定义域名
- 在 Custom domain 中填写：xynong.ccwu.cc
- 点击 Save

## 4. 配置域名解析
在你的域名服务商后台添加解析：
- 类型：A
- 主机名：@
- 值：185.199.108.153 / 185.199.109.153 / 185.199.110.153 / 185.199.111.153

如果你使用 Cloudflare，请设置为“DNS only（灰云）”，不要启用代理（orange cloud）。

## 5. 访问地址
部署完成后访问：
https://xynong.ccwu.cc
