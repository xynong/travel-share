# 当前域名解析建议

你的域名是：xynong.ccwu.cc

## 结论
建议使用 A 记录，而不是 CNAME。

## 该怎么配
在域名服务商后台新增 4 条 A 记录：
- 185.199.108.153
- 185.199.109.153
- 185.199.110.153
- 185.199.111.153

### 字段说明
- 类型：A
- 主机名 / 名称：@
- 值 / 目标：上面的 IP 地址
- TTL：保持默认即可

如果你使用 Cloudflare，请务必把这些 A 记录设置为“DNS only（灰云）”，不要开启代理（orange cloud）。

## 说明
- @ 表示根域名，也就是 xynong.ccwu.cc
- GitHub Pages 对根域名最稳妥的方式是 A 记录
- 如果你想配 www 子域名，再单独加一条 CNAME
