# 最短发布步骤（安装 Git 后）

仓库地址：https://github.com/xynong/travel-share.git
域名：xynong.ccwu.cc

## 一、安装 Git

1. 下载并安装 Git for Windows：https://git-scm.com/download/win
2. 安装完成后，打开 PowerShell，输入：

```powershell
git --version
```

如果能看到版本号，说明安装成功。

## 二、进入仓库并配置身份

```powershell
cd C:\Users\xieyong\GitHub\travel-share
git config --global user.name "xieyong"
git config --global user.email "your@email.com"
```

## 三、提交并推送到 GitHub

```powershell
git status
git add .
git commit -m "更新旅游官网"
git branch -M main
git remote add origin https://github.com/xynong/travel-share.git
git push -u origin main
```

> 如果这个仓库已经有远程地址，跳过 `git remote add origin ...` 这一行。

## 四、开启 GitHub Pages

1. 打开 GitHub 仓库页面。
2. 进入 Settings > Pages。
3. Source 选择 Deploy from a branch。
4. Branch 选择 `main`，Folder 选择 `/root`。
5. 保存后等待部署完成。

## 五、绑定自定义域名

1. 在 Pages 设置中输入：`xynong.ccwu.cc`
2. 到你的域名解析服务商中配置解析：
   - 选择 CNAME 或 A 记录
   - 将 `xynong.ccwu.cc` 指向 GitHub Pages
3. 等待 DNS 生效后访问：
   - https://xynong.ccwu.cc
