# 旅游开机包

项目仓库：https://github.com/xynong/travel-share.git
自定义域名：xynong.ccwu.cc

## 一键上传命令

```bash
git add .
git commit -m "更新旅游官网内容"
git branch -M main
git remote add origin https://github.com/xynong/travel-share.git
git push -u origin main
```

## 如果是首次上传

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/xynong/travel-share.git
git push -u origin main
```

## GitHub Pages 设置

1. 打开 GitHub 仓库页面。
2. 进入 Settings > Pages。
3. Source 选择 Deploy from a branch。
4. Branch 选择 main，Folder 选择 /root。
5. 保存后等待部署完成。

## 自定义域名设置

1. 在 Pages 设置中填写 xynong.ccwu.cc。
2. 在域名解析服务商处添加 CNAME 或 A 记录。
3. 等待 DNS 生效后访问站点。
