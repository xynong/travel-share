# 电脑版操作手册：发布到 GitHub Pages

仓库地址：https://github.com/xynong/travel-share.git
自定义域名：xynong.ccwu.cc

## 第一步：安装 Git
如果电脑还没装 Git：
1. 下载安装 Git for Windows
2. 安装完成后打开 PowerShell
3. 输入：

```powershell
git --version
```

如果看到版本号，说明安装成功。

## 第二步：进入项目目录
```powershell
cd C:\Users\xieyong\GitHub\travel-share
```

## 第三步：设置 Git 身份
```powershell
git config --global user.name "xieyong"
git config --global user.email "your@email.com"
```

## 第四步：提交并推送
```powershell
git status
git add .
git commit -m "更新旅游官网"
git branch -M main
git push -u origin main
```

## 第五步：在 GitHub 开启 Pages
1. 打开 GitHub 仓库页面
2. 点击 Settings
3. 左侧选择 Pages
4. Source 选 Deploy from a branch
5. Branch 选 main，Folder 选 /root
6. 点击 Save

## 第六步：绑定自定义域名
1. 在 Pages 页面中找到 Custom domain
2. 填写：
   xynong.ccwu.cc
3. 点击 Save

## 第七步：配置域名解析
在域名服务商后台添加解析：
- 类型：CNAME 或 A 记录
- 域名：xynong.ccwu.cc
- 指向：GitHub Pages 地址

## 第八步：等待生效
等一段时间后访问：
https://xynong.ccwu.cc
